== Description ==
 افزونه درگاه پرداخت پی‌پینگ برای افزونه فرم تماس 7

== Installation ==
1. فایل افزونه را در مسیر \'/wp-content/plugins/\' آپلود کنید
2. افزونه را از طریق منوی \'افزونه ها\' در وردپرس فعال کنید
